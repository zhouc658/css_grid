String Figure

Crystal Zhou 2024

Watercolor on Paper, 18 x 24 in

String Figure is a game that I have played since I was little with my friends. It is always fun and interesting to see how far we can go with this game and figure out how to get from one figure to the next. As long as you have a string you can play it anywhere, and in this case, it is a drawstring from my sweatpants.
